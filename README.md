Flower Shop Website
This project is a website for a small flower shop. It's built using HTML, SCSS, and JavaScript, and Gulp is used to enhance development efficiency. The website is designed to promote small flower shop and provide information about the main products and services offered.

Features:

Responsive design to ensure accessibility on all devices.
A product catalog with descriptions and prices.
Contact information for customers.
Technologies Used:

HTML
SCSS
JavaScript
Gulp
